# 스튜디오를 시작하며 — 게으른 너구리의 첫 커밋

왜 하필 게으른 너구리인지, 스튜디오를 시작한 이야기입니다.

## 게으름이라는 태도

게으름은 대충이 아니라, 서두르지 않는다는 뜻입니다. 오래 갈 것을 만듭니다.

```bash
$ git init
$ git commit -m "first commit: 게으른 너구리"
$ git push origin main
```

## 첫 커밋

모든 것은 하나의 커밋에서 시작했습니다. 이제 하나씩 쌓아갑니다.
