# Trashpanda CLI

A CLI toolkit that lets developers **ship straight to production with one command**.

## Features
- Deploy a static site with a single `git push`
- Per-environment config management
- Auto-generated GitHub Actions workflows

## Tech
Node.js · TypeScript · GitHub Actions

## Status
Open-sourced and actively maintained.
