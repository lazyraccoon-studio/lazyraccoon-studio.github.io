# Trashpanda CLI

개발자가 **한 줄 명령으로 배포까지** 끝내도록 돕는 CLI 툴킷입니다.

## 기능
- `git push` 한 번으로 정적 사이트 배포
- 환경별 설정 관리
- GitHub Actions 워크플로 자동 생성

## 기술
Node.js · TypeScript · GitHub Actions

## 상태
오픈소스로 공개해 운영 중입니다.
