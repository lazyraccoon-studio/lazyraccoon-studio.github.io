# Nightshift

한 스타트업의 **실시간 운영 대시보드**를 8주 만에 설계·구축한 외주 프로젝트입니다.

## 역할
- 실시간 데이터 파이프라인 설계
- 대시보드 프론트엔드 개발
- 배포 자동화 구축

## 기술
React · Go · PostgreSQL · WebSocket

## 결과
운영팀의 야간 대응 시간을 크게 단축했습니다.
